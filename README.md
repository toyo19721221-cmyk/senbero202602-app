# 糸島センベロ 2026 スタンプラリーアプリ

糸島エリアの飲食店活性化を目的とした、ブラウザで動作するモバイル特化型PWAスタンプラリーアプリです。

## 🚀 開発背景
地域の飲食店を回遊する「センベロイベント」において、紙のスタンプ台紙による紛失リスクや、制作コストを削減するために開発しました。
ユーザーが「アプリインストール不要」で、QRコードを読み取るだけで即座に参加できる体験を重視しています。

## ✨ 特徴・こだわり
- **PWA (Progressive Web Apps) 対応**: ホーム画面に追加することで、ネイティブアプリのような挙動を実現。
- **カメラ連携スキャン**: `jsQR` ライブラリを採用し、ブラウザ上で高速なQR読み取りが可能。
- **オフライン体験**: `LocalStorage` を活用し、通信環境が不安定な店内でもスタンプ情報を保持。
- **モバイルファーストUI**: Tailwind CSS を用い、片手で操作しやすい直感的なデザインを構築。

## 🛠 使用技術
- **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla JS)
- **Library**: [jsQR](https://github.com/cozmo/jsQR) (QRコード解析)
- **Animation**: [canvas-confetti](https://www.npmjs.com/package/canvas-confetti) (達成演出)
- **Platform**: GitHub Pages

## 📱 使い方
1. アプリを開き「ホーム画面に追加」をタップ（推奨）。
2. 加盟店でセンベロを注文し、店内のQRコードをスキャン。
3. 全25店舗のスタンプを集めると、コンプリート画面が表示されます。
